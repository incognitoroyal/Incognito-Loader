# Incognito-Loader
Just a simple C++ (Visual Only) Loader for Incognito.cc (Not official) well a practice version of it. First time using windows api so got ahead. You can also change the name of the loader when running it if you want too but the main stuff like ASCII will stay the same.

# How to install
Just download the exe release, always scan online files with a tool like virus total before using them: virustotal.com/gui (Dm me on discord if you think I should add source or something)
